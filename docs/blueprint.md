# **App Name**: SoleDelights Landing

## Core Features:

- Homepage with Photo and Description: Showcase Sole's photo and a welcoming description of her pastry business.
- Gallery: Display a photo gallery of Sole's cake creations to attract customers.
- "Quiero mi torta" Button: Call-to-action button that leads users to the cake order form.
- Cake Size and Servings Selector: Allow users to select cake sizes with corresponding serving suggestions using clear icons.
- Flavor (Cake) Selection: Provide a list of cake flavor options (Vanilla, Chocolate, etc.).
- Filling and Coverage Options: Enable users to select filling and coverage types (Buttercream, Ganache).
- Customization Textarea: Provide a textarea where customers can describe the cake's theme, colors, or desired message.
- Payment and Delivery Form: Form input fields for customer name, contact (WhatsApp), desired delivery date, and payment method selection.
- Dynamic Payment Option Display: Display specific payment instructions based on whether the user chooses Efectivo/Zelle or PagoMovil, including an upload field for a payment screenshot when choosing PagoMovil.

## Style Guidelines:

- Primary color: Lila (#D989F7) for navbar and main buttons, providing a magical yet professional aesthetic.
- Accent color: Bright Yellow (#FFF200) used for star icons and to draw attention to key elements, providing a touch of Kawaii.
- Background: Clean White to maintain a professional and bright presentation.
- Font: 'Poppins', a geometric sans-serif for a modern, fashionable and professional feel.
- Use "kawaii" style icons, similar to Instagram story highlight icons, to enhance visual appeal while retaining a professional appearance.
- Employ a clean, organized layout with ample spacing to maintain professionalism while integrating 'kawaii' elements smoothly.
- Use subtle animations on button hover states to indicate interactivity and provide visual feedback.
